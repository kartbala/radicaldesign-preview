DESCRIPTION

About this product

Finally a Hiking Trailer that lets you transport any bag, duffel or waterproof Ortlieb but also equipment, wood, stones and more within a width of 30cm (11.8") max. Its multiple lashing points make it easy to securely strap things in. The Cargo Hiking Trailer version also enables to make long distance hiking journeys and after setting camp do smaller trips from that point with the Backpack.
