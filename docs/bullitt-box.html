DESCRIPTION

About this product

To give the Bullitt Bag V2 a more rigid shape we developed the Bullitt Bag Box Insert. This system of foldable plates of 5mm honeycomb board creates a box structure which optimizes the space on the cargo area of the Bullitt Bike. The box makes it easier to load and organize the bag, as the available space is already opened up. When the bag is empty, the box fold back to a small package which lays flat on the bottom. This allows you to still freely use the Bullitt Bag V2 as a loose bag.

The box insert fixes to the Bullitt Bag V2 with a series of velcro straps which are designed for ease of use. This allows you to convert the cargo-bag to a cargo-box within 1 minute.
