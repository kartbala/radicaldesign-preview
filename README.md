# radicaldesign-preview

Unofficial redesign preview of radicaldesign.com, built as a gift for
Radical Design Manufacturing BV (Gasselternijveen, NL) by a customer.
Not the official site; not affiliated with or endorsed by Radical Design.
Product photos and text belong to Radical Design Manufacturing BV.

Static site lives in `public/`; served via GitHub Pages. `robots.txt`
blocks indexing so this preview never competes with the real site in
search results.
